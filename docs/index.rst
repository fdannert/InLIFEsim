===============================
InLIFEsim Documentation
===============================
.. _LIFEsim: https://lifesim.readthedocs.io

Welcome to the InLIFEsim documentation! InLIFEsim is a simulator built to analyze the impact of coherent perturbations on exoplanet detections with nulling interferometers. 
It extends the capabilities of `LIFEsim`_.
